# sales-analysis-powerbi
Interactive Power BI Sales Dashboard designed to analyze revenue, profit, quantity sold, shipping performance, and category-wise sales trends. The dashboard includes KPI cards, slicers, bookmarks, navigation buttons, and dynamic visualizations to provide meaningful business insights and improve decision-making.


# Sales Performance Dashboard - Power BI

## Project Overview
This Power BI dashboard analyzes sales performance across different categories, countries, and shipping modes.

The dashboard helps identify:
- Total Revenue
- Total Profit
- Quantity Sold
- Top Performing Countries
- Revenue Trends Over Time
- Shipping Analysis

---

## Tools Used
- Power BI
- Power Query
- DAX
- Excel / CSV Dataset

---

## Features
- KPI Cards
- Interactive Slicers
- Bookmarks & Navigation Buttons
- Top 10 Country Analysis
- Revenue Trend Analysis
- Ship Mode Analysis
- Category Filtering

---

## KPIs Used
- Total Revenue
- Total Profit
- Total Quantity Sold
